```scm
(define (A x y)
  (cond ((= y 0) 0)
        ((= x 0) (* 2 y))
        ((= y 1) 2)
        (else (A (- x 1)
                 (A x (- y 1))))))
```

`(A 1 10)` 值为 `2^10`

`(A 2 4)` 值为 `(((2^2)^2)^2)^2`，即 65536

`(A 3 3)` 值为 65536

`(define (f n) (A 0 n))` 为 `2n`

`(define (g n) (A 1 n))` 为 `2^n`

`(define (h n) (A 2 n))` 为 2 一直平方 n 次
